# 更新日志

这里记录 MediaSlim 面向用户的正式版本变化。日期以 GitHub Release 的正式发布日期为准，最新版本位于最上方。

## [1.0.0] - 2026-07-21

首个正式 macOS 版本。

### 新增

- 支持拖放或手动选择 PNG、JPG、JPEG 图片，并支持批量处理。
- 支持直接添加文件夹，自动查找其中可压缩的图片。
- PNG 与 JPEG 均在本机完成压缩，无需上传原图。
- 将更小的压缩副本保存到原文件旁的 `Compressed` 文件夹，不覆盖原图。
- 展示批处理进度、单张图片结果、压缩前后体积和节省比例。
- 支持取消处理、在 Finder 中定位输出文件，以及简体中文和英文界面。
- 提供经过签名与公证的 DMG 安装包，并支持 Sparkle 应用内更新。
